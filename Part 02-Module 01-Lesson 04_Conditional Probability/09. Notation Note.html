09. Notation Note

Notation Note

This lesson will use some notation that you may not be familiar with. The table below explains some of that notation.

Symbol Usage Interpretation
P P(A) "The probability of event A occurring"
\lnot P(\lnot A) "The probability of event A not occurring"
\mid P(A \mid B) "The probability of event A occurring given that event B occurs"
