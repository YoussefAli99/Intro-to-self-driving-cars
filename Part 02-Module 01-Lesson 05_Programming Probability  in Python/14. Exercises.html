Exercises

It may take a few attempts to answer an exercise correctly, and that's okay. If, after multiple attempts, you are still struggling to correctly solve an exercise, we will try to include solution code. Feel free to read through the solution to understand how it works. Then (without copy/pasting) try to go back to the problem and implement that solution yourself before moving on!
